história da Web

A Web foi inventada por Berners-Lee em 1989, enquanto trabalhava no CERN. A ideia original era criar um sistema de informação global que permitisse o compartilhamento automático de informações entre cientistas em universidades e institutos ao redor do mundo. O primeiro servidor e navegador da Web foram criados em 1990, e o primeiro site da Web foi lançado em 1991. Desde então, a Web cresceu e se tornou uma ferramenta essencial para a comunicação e o comércio em todo o mundo.

A Web é baseada em hipertexto, que permite que os usuários naveguem por documentos interconectados usando links. O HTML é a linguagem usada para criar documentos da Web, e o HTML5 é a versão mais recente dessa linguagem. O HTML5 inclui muitos recursos novos, como suporte para vídeo e áudio incorporados, gráficos vetoriais escaláveis, armazenamento local e muito mais.

Se você está começando agora sua rotina jornada na programação, aprender HTML5 é um ótimo lugar para começar. Com o HTML5, você pode criar sites interativos e responsivos que funcionam em todos os dispositivos. Boa sorte nos seus estudos!

O que são Clients

Um cliente é um computador ou programa que, como parte de sua operação, depende do envio de uma solicitação para outro programa ou hardware de computador ou software que acessa um serviço disponibilizado por um servidor (que pode ou não estar localizado em outro computador). Em outras palavras, um cliente é uma aplicação , como um navegador da Web, que opera em um computador, telefone celular ou qualquer outro dispositivo e ajuda os usuários a realizar várias operações, como fazer uma solicitação ou qualquer outra tarefa. Por exemplo, navegadores da Web são clientes que se conectam a servidores da Web e recuperam páginas da Web para exibição. Os clientes também podem ser programas que executam o software do cliente ou usuários que usam o software do cliente.

Em resumo, quando algo está no lado do servidor, basicamente significa que ele será executado no servidor ou em uma máquina remota. Por outro lado, quando algo está no lado do cliente, significa que ele é executado em seu aplicativo de computador. Quando se trata de desenvolvimento web, scripts e programas do lado do cliente são incorporados na página da web do cliente e processados no navegador. Por outro lado, quando se trata de processamento do lado do servidor, ele ocorre nos servidores onde bancos de dados e arquivos permanentes são armazenados.

O que são Servers

Um Servidor é um computador ou programa que fornece serviços, dados ou recursos para outros computadores, conhecidos como clientes, por meio de uma rede. Em outras palavras, um servidor é um programa que executa em um computador ou dispositivo e ajuda os usuários a realizar várias operações, como armazenar e gerenciar arquivos, executar aplicativos, hospedar sites e muito mais. Por exemplo, servidores da Web são programas que se conectam a clientes da Web e fornecem páginas da Web para exibição. Os servidores também podem ser programas que executam o software do servidor ou usuários que usam o software do servidor.

Em resumo, quando algo está no lado do servidor, basicamente significa que ele será executado no servidor ou em uma máquina remota. Por outro lado, quando algo está no lado do cliente, significa que ele é executado em seu aplicativo de computador. Quando se trata de desenvolvimento web, scripts e programas do lado do cliente são incorporados na página da web do cliente e processados no navegador. Por outro lado, quando se trata de processamento do lado do servidor, ele ocorre nos servidores onde bancos de dados e arquivos permanentes são armazenados.

Linguagens de Programação

Linguagens de programação são idiomas que os programadores usam para escrever programas de computador, scripts ou outros conjuntos de instruções para que os computadores executem. Cada linguagem de programação tem sua própria sintaxe, que é como a gramática de uma língua. Existem muitas linguagens de programação diferentes, como C, C++, Java, Python e muitas outras. Cada linguagem tem suas próprias vantagens e desvantagens, e é importante escolher a linguagem certa para o trabalho que você está fazendo. Aprender uma linguagem de programação pode ser um pouco difícil no começo, mas com prática e perseverança, você pode se tornar um programador habilidoso em pouco tempo!

Estrutura Básica do HTML e suas TAGS

A estrutura de um arquivo HTML é composta por tags que são usadas para marcar diferentes partes do conteúdo. Cada tag é escrita entre colchetes angulares < > e geralmente vem em pares de abertura e fechamento. A tag de abertura é escrita com o nome da tag, enquanto a tag de fechamento tem o mesmo nome, mas com uma barra antes do nome da tag. Por exemplo, a tag "p" é usada para marcar um parágrafo e a tag "/p" é usada para fechar o parágrafo.

Aqui está um exemplo básico de estrutura de arquivo HTML:

        
            <!DOCTYPE html>
            <html>
                <head>
                    <title>Título da página</title>
                </head>
                <body>
                    <h1>Título da página</h1>
                    <p>Este é um parágrafo.</p>
                </body>
            </html>

Neste exemplo, temos as seguintes tags:

  1. "!DOCTYPE html": Define o tipo de documento como HTML5.
  2. "html": Marca o início e o fim do documento HTML.
  3. "head": Contém informações sobre o documento, como o título da página.
  4. "title": Define o título da página que aparece na barra de título do navegador.
  5. "body": Contém todo o conteúdo visível da página.
  6. "h1": Marca um cabeçalho principal.
  7. "p": Marca um parágrafo.

Listas e Atributos

Listas ordenadas e listas não ordenadas são estruturas usadas em HTML para agrupar itens relacionados.

Listas não ordenadas são usadas quando a ordem dos itens não importa. Os itens são marcados com pontos ou outros símbolos. Para criar uma lista não ordenada, usamos a tag "ul". Cada item da lista é declarado dentro da tag "li"

Listas ordenadas são usadas quando a ordem dos itens é importante. Os itens são numerados sequencialmente. Para criar uma lista ordenada, usamos a tag "ol". Cada item da lista é declarado dentro da tag "li"
